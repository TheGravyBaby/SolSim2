Welcome to Sol Sim!

Controls
1. To add a body, right click empty space.
2. To move a body, select and drag.
3. To change velocity, toggle vectors then drag.
4. To move camera, select empty space and drag.
5. To zoom in and out, use the scroll wheel.

You can also select a body from the dropdown list, and edit it's properties manually using the text fields.

There are multiple pre built systems you can select from the 'Star Systems' drop down. If you make one you particularly like, you can download or upload the JSON file containing all the body data.
