Welcome to Sol Sim!

Controls
1. To add a body, right click empty space and a body will appear.

2. To select a body, click on it or use the "Orbital Body" dropdown.

3. Once selected you can edit the properties of the body on the right side (pause first)

4. To delete a body, press the Del key after a body has been selected

5. To easily change the position of a body, simply drag the body where you want it to be on the map.

6. To change the velocity of a body, enable velocity vectors and drag the vector. Or use the property grid on the right.

7. To delete a body, select the body from the d 8. You can move around the map by dragging empty space, and by using the scroll wheel to zoom.


There are multiple pre built systems you can select from the 'Star Systems' drop down. If you make one you particularly like, you can download or upload the JSON file containing all the body data.
